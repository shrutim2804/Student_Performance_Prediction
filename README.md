# 🎓 Student Performance Prediction System

## Objective
Predict student academic performance (Low/Medium/High) using engagement and behavioral features.

## Dataset
- Source: Kaggle - xAPI Educational Data Mining Dataset
- Records: 480 students, 16 features

## ML Models Used
- Logistic Regression
- Random Forest
- XGBoost (Best Model)

## Tech Stack
- Python 3.13
- Scikit-learn, XGBoost, Pandas, NumPy
- Flask (Web App)
- Matplotlib, Seaborn (Visualization)

## How to Run
```bash
cd Flask_App
..\venv\Scripts\python.exe app.py
```
Open http://127.0.0.1:5000 in browser.

## Project Structure
- Dataset/ — raw and processed data
- Notebook/ — Jupyter notebooks for each phase
- Model/ — saved ML models
- Flask_App/ — web application
- Documentation/ — charts and metrics
